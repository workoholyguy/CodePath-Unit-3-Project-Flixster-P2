# Project 3 - project3-flixster-p2

Submitted by: Omar Madjitov

Flixster is an app that introduces networking calls to an application. Flixster Displays the List of movies that are in the theaters right now, as well as the Details about those movies. Flister allows the user to switch the tabs in order to see the list of movies as a Collection view.

Time spent: 10 hours spent in total

- [x] Implement network call and table view to display a list of movies currently playing (title, description, image)
